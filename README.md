# ShopWave Frontend

React 18 + TypeScript SPA for the ShopWave e-commerce platform.

## Tech Stack
- React 18 + TypeScript 5
- Vite 5 (build tool)
- TanStack Query v5 (data fetching)
- Zustand (global state)
- React Router v6
- Tailwind CSS + shadcn/ui
- Vitest + React Testing Library

## Quick Start
```bash
cp .env.example .env.local
npm install
npm run dev
```
